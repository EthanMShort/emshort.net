SANDBOX // SELF-HOSTED

The Home Lab

This is my personal playground for testing enterprise tech and hosting services for my home. It’s a hybrid setup that lets me simulate a corporate Windows environment while running fun, modern apps in Docker.

The core runs on Unraid for flexible storage and containers (like Plex and HomeAssistant), while a separate Windows Server environment handles Active Directory and DNS. The entire network is secured and segmented using Ubiquiti UniFi gear.

The Stack

Compute & Storage

Unraid OS | Lenovo ThinkStation P520 | 5TB HDD (RAID1)

My primary server runs Unraid to handle NAS duties and Docker containers. I also plan to run Hyper-V instances for Windows Server 2022 to practice AD administration.

GATEWAY01

Network Infrastructure

Ubiquiti UniFi

A modest UniFi stack (UCG Max) managing multiple VLANs. This keeps my trusted lab devices separate from the IoT chaos.

View Network Details
  • Gateway: UniFi Cloud Gateway Max
  • Switching: Using the 2.5 GbE RJ45 LAN Ports on the UCG Max for now
  • Wireless: Unifi U7 Pro for 2.4 GHz, 5 GHz, and 6 GHz connectivity
  • Segmentation: Main (Standard Network), HomeLab (Sepreate NAS01, HomeAssistant, and other appliances), Guest, IoT Network (For Smart Plugs and Lights), Isolated (For dockers that I only want to have acces to external internet, nothing internal)
